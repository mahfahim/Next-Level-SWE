### TypeScript প্রজেক্ট শুরু থেকে রান করার সম্পূর্ণ টিউটোরিয়াল

আপনার টাইপস্ক্রিপ্ট প্রজেক্ট শুরু থেকে শেষ পর্যন্ত কীভাবে তৈরি করবেন, সেটি ধাপে ধাপে দেখানো হলো।

---

### **১. TypeScript ইনস্টল করা**

প্রথমে আপনার সিস্টেমে TypeScript ইনস্টল করতে হবে। যদি আপনি Node.js ব্যবহার করেন, তাহলে `npm` দিয়ে TypeScript ইনস্টল করতে পারেন।

```bash
npm install -g typescript
```

এটি টাইপস্ক্রিপ্ট কম্পাইলার (`tsc`) কে গ্লোবালি ইনস্টল করবে।

---

### **২. টাইপস্ক্রিপ্ট প্রজেক্ট ইনিশিয়েট করা (`tsc --init`)**

এখন প্রজেক্টের জন্য `tsconfig.json` ফাইল তৈরি করতে হবে। এই ফাইলটি টাইপস্ক্রিপ্ট কম্পাইলারের কনফিগারেশন ধারণ করে, যাতে আপনি কোড কম্পাইল করার সময় কাস্টম সেটিংস ব্যবহার করতে পারেন।

প্রজেক্ট ফোল্ডারে গিয়ে নিচের কমান্ডটি চালান:

```bash
tsc --init
```

এটা একটি `tsconfig.json` ফাইল তৈরি করবে, যেখানে আপনি কোড কম্পাইল করার জন্য প্রয়োজনীয় কনফিগারেশন পাবেন। উদাহরণস্বরূপ, ফাইলটি এমন দেখাবে:

```json
{
  "compilerOptions": {
    "target": "es5",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "outDir": "./dist"
  },
  "include": [
    "src/**/*.ts"
  ]
}
```

এখানে `outDir` দিয়ে আপনি নির্ধারণ করতে পারেন যে কম্পাইলড ফাইলগুলো কোথায় যাবে। আমি এখানে `./dist` ফোল্ডার ব্যবহার করেছি।

---

### **৩. TypeScript কোড লেখা**

এখন টাইপস্ক্রিপ্ট ফাইল তৈরি করতে হবে। আপনি আপনার কোড লিখতে পারেন `src` নামক একটি ফোল্ডারে।

উদাহরণস্বরূপ, `src/index.ts` নামক একটি টাইপস্ক্রিপ্ট ফাইল তৈরি করুন:

```typescript
// src/index.ts
let message: string = "Hello, TypeScript!";
console.log(message);
```

এখানে আপনি স্রেফ একটি সিম্পল মেসেজ আউটপুট করতে লিখেছেন।

---

### **৪. টাইপস্ক্রিপ্ট কোড কম্পাইল করা**

এখন আপনি `tsc` কমান্ড দিয়ে আপনার টাইপস্ক্রিপ্ট কোড কম্পাইল করতে পারবেন। যদি আপনি `tsconfig.json` ফাইল ব্যবহার করেন, তাহলে শুধু নিচের কমান্ডটি চালালেই হবে:

```bash
tsc
```

এটা আপনার টাইপস্ক্রিপ্ট কোডকে কম্পাইল করে `dist` ফোল্ডারে জাভাস্ক্রিপ্ট ফাইল তৈরি করবে (যেহেতু আপনি `outDir` হিসেবে `dist` নির্ধারণ করেছেন)। এই ক্ষেত্রে, `src/index.ts` কম্পাইল হয়ে `dist/index.js` হবে।

---

### **৫. JavaScript কোড রান করা**

আপনি এখন আপনার কম্পাইল করা `.js` ফাইল রান করতে পারবেন। নোড (Node.js) ব্যবহার করে এটি করা যায়:

```bash
node dist/index.js
```

এটি চালালে আউটপুট হবে:

```
Hello, TypeScript!
```

---

### **৬. অটোমেটিক কম্পাইল করার জন্য `--watch` ব্যবহার করা**

যদি আপনি চান যে আপনার টাইপস্ক্রিপ্ট কোড পরিবর্তন হলে অটোমেটিকভাবে কম্পাইল হয়ে যায়, তাহলে `--watch` ফ্ল্যাগ ব্যবহার করতে পারেন।

এটি চালালে আপনি যতবার `.ts` ফাইল পরিবর্তন করবেন, ততবার তা অটোমেটিকভাবে কম্পাইল হয়ে যাবে।

```bash
tsc --watch
```

এটি আপনার ফাইলগুলোর পরিবর্তন মনিটর করবে এবং প্রতিবার পরিবর্তন হলে কম্পাইল করে ফেলবে।

---

### **৭. টেস্ট করা**

আপনি এখন যে কোন সময় আপনার `.ts` ফাইলের মধ্যে পরিবর্তন করে আবার রান করতে পারবেন:

1. `index.ts` ফাইলের মধ্যে কিছু পরিবর্তন করুন, যেমন:

```typescript
// src/index.ts
let message: string = "TypeScript is awesome!";
console.log(message);
```

2. তারপর আবার `tsc --watch` চালান, এবং `node dist/index.js` দিয়ে নতুন আউটপুট দেখুন।

এখন আউটপুট হবে:

```
TypeScript is awesome!
```

---

### **৮. (ঐচ্ছিক) আরও কাস্টমাইজেশন**

`tsconfig.json` ফাইলের মধ্যে আপনি আরও কাস্টম সেটিংস করতে পারেন। উদাহরণস্বরূপ:

* `strict`: টাইপ সেফটির জন্য সব রকমের চেক চালাবে।
* `target`: কোড কোন ECMAScript ভার্সনে কম্পাইল হবে (যেমন `es5`, `es6` ইত্যাদি)।
* `module`: মডিউল সিস্টেম (যেমন `commonjs` বা `es6` মডিউল সিস্টেম)।

আপনার প্রজেক্টের প্রয়োজন অনুযায়ী আপনি এই কনফিগারেশন পরিবর্তন করতে পারেন।

---

এভাবে আপনি **TypeScript** প্রজেক্ট শুরু থেকে রান করতে পারবেন।
